Skill 2: HTML

Introduction to HTML

HTML is the standard language used to structure content on the web. It acts as the foundation of web pages, defining their structure and meaning. HTML works hand-in-hand with CSS for styling and JavaScript for interactivity.

What HTML Can Do

  • Build Structure: HTML creates the skeleton of a web page, including headers, paragraphs, lists, and links.
  • Embed Media: It supports images, videos, and audio files.
  • Interactive Elements: Use forms, buttons, and inputs to make web pages interactive.
  • SEO Optimization: Semantic HTML helps search engines understand the content, improving website visibility.

Fun Facts About HTML

  • The first version of HTML was created by Tim Berners-Lee in 1991.
  • HTML5 was introduced in 2014 and brought major improvements.
  • HTML is used in every single website you visit!
